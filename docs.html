API Documentation

The Shadon API gives you programmatic access to breach data, OSINT sources, IP intelligence, and domain analysis. API access is available on the Pro plan (1,000 requests/day) and Enterprise plans.

Overview

All API requests are made to the base URL:

https://api.shadon.io/v1

All responses are returned as JSON. Requests must include your API key in the header. HTTPS is required for all requests.

Authentication

Include your API key in the X-API-Key request header on every request. You can find your API key in your dashboard.

curl -H "X-API-Key: shdn_your_api_key_here" \
  https://api.shadon.io/v1/search?q=test@example.com&type=email
Note: Never expose your API key in client-side code. Use it server-side only. Rotate your key immediately if you believe it has been compromised.

Rate limits depend on your plan:

Rate limit headers are included in every response: X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset.

Search Endpoint

GET/v1/search

Universal search endpoint. Shadon auto-detects the query type or you can specify it explicitly.

ParameterTypeRequiredDescription
qstringrequiredThe search query (email, username, IP, domain, or hash)
typestringoptionalForce query type: email, username, phone, ip, domain, hash
sourcesstringoptionalComma-separated source filter: breaches, stealer, osint
 
GET /v1/search?q=user@example.com&type=email HTTP/1.1
Host: api.shadon.io
X-API-Key: shdn_your_key

Email Lookup

GET/v1/email/{email}

Look up an email address across all breach databases and stealer log sources. Returns found breaches, exposed passwords (Pro), and associated accounts.

curl -H "X-API-Key: shdn_your_key" \
  https://api.shadon.io/v1/email/user@example.com

Domain Analysis

GET/v1/domain/{domain}

Returns DNS records, WHOIS/RDAP data, SSL certificate details, and security reputation scores.

curl -H "X-API-Key: shdn_your_key" \
  https://api.shadon.io/v1/domain/example.com

Hash / File Analysis

GET/v1/hash/{hash}
POST/v1/file

Search a file hash (MD5, SHA1, SHA256) across 70+ antivirus engines, or upload a file for direct analysis.

# Search by hash
curl -H "X-API-Key: shdn_your_key" \
  https://api.shadon.io/v1/hash/d41d8cd98f00b204e9800998ecf8427e

# Upload file (POST, multipart)
curl -H "X-API-Key: shdn_your_key" \
  -F "file=@/path/to/file.exe" \
  https://api.shadon.io/v1/file

Response Format

All responses follow a consistent envelope:

{
  "success": true | false,
  "query": "...",
  "type": "email | username | ip | domain | hash",
  "found": true | false,
  "results": { ... },
  "meta": {
    "total_sources": 0,
    "query_time_ms": 0,
    "plan": "pro"
  }
}

Error Codes

CodeMeaning
401Missing or invalid API key
403Feature requires Pro plan
422Invalid query format
429Rate limit exceeded
500Internal server error

Error responses include a message field with a human-readable description of the error.

{
  "success": false,
  "error": 429,
  "message": "Daily rate limit exceeded. Resets in 3h 22m.",
  "reset_at": "2025-01-15T00:00:00Z"
}
